# projeto-bikcraft

[Vizualização do site](https://jeferson-m.github.io/projeto-bikcraft/)

Projeto de conclusão do curso de HTML & CSS da Origamid, colocando em prática a teoria 
aprendida sobre a estrutura do HTML, CSS e Design das páginas definindo seus estilos,
espaçamentos, tipografias e cores. Respeitando a semântica e facilitando a leitura 
criando um código mais limpo. Foi utilizado todos os recursos mais importantes visto no curso
como por exemplo o uso de Grid Layout e Flex Box para criação de vários elementos interessantes.
Priorizando a responsividade em diferentes tamanhos de telas utilizando o Media Queries.

*O site é apenas um projeto fictício sem qualquer intenção de representar um modelo totalmente real.*

![image](https://github.com/jeferson-m/projeto-bikcraft/assets/106871718/c79b2584-3bff-4b1b-92ab-da2816f64ac7)

Foi utilizado a diretriz @import para carregar arquivos externos para o CSS e melhorar a
organização do CSS facilitando a busca do código caso precise realizar qualquer ajuste.

![image](https://github.com/jeferson-m/projeto-bikcraft/assets/106871718/804bf6e7-46c6-4ba3-86aa-2dda297ccea3)

Criado também um CSS mais limpo reutilizando componentes já prontos para serem reutilizados, como
o container (tamanho total do layout da página), cores, tipografias, padrão de botões, etc.
